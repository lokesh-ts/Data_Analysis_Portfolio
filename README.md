# Data Analysis Portfolio

This repository contains 10 Python projects demonstrating data analysis skills using Pandas, NumPy, and Matplotlib.
Each project includes analysis scripts, datasets, and visualizations.

## Projects Included

1. **Traffic Flow Analysis** – Analyze traffic patterns and daily/monthly traffic counts.
2. **College Attendance Analytics** – Track student attendance and visualize trends.
3. **Internet Usage Analysis** – Study internet usage patterns and peak times.
4. **Electricity Consumption Analysis** – Analyze household electricity usage.
5. **Public Transport Ticket Revenue Analysis** – Track daily and monthly ticket revenue.
6. **Website Traffic Analysis** – Analyze website visits and identify peak days.
7. **Battery Performance Analysis** – Monitor battery health and usage trends.
8. **Air Quality Index (AQI) Analysis** – Analyze daily/monthly AQI trends.
9. **Sports Performance Analytics** – Evaluate player and team performance.
10. **Movie Box Office Collection Analysis** – Track revenue trends and top-grossing movies.

## Tools & Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib
