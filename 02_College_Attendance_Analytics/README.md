# College Attendance Analytics

This project analyzes college student attendance data to identify attendance patterns,
calculate attendance percentages, and find students at risk due to low attendance.

## Objectives
- Analyze daily and monthly attendance trends
- Calculate attendance percentage for each student
- Identify students with attendance below required threshold
- Visualize attendance patterns using charts

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains student attendance records including:
- Student ID
- Student Name
- Date
- Attendance Status (Present/Absent)

## Outcome
This analysis helps institutions monitor attendance, improve student engagement,
and take early action for students with low attendance.
